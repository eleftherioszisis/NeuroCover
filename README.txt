NeuroCover
-----------


This Python module includes: 

* Toolkits to compute the volume coverage of a morphology as a function of an inflation radius. 

=============
Supported OS
=============

Ubuntu : 12.0, 14.04

====================
Required Dependencies
====================

Python : 2.7


====================
Installation
====================

# Create virtual environment
virtualenv venv
# Activate venv
. ./venv
# Pip install package with dependencies
pip install -e NeuroCover
